DumpedKnowledge - AI Research Companion

I was overwhelmed reading many websites on a research project I was doing, so I decided to build my own FullStack Research companion. Below is how it transforms simple URL submissions into a queryable, intelligent knowledge repository.

üîß Backend Technologies

  • FastAPI - Python web framework
  • SQLite - Database for user accounts and article metadata
  • SQLAlchemy - ORM for database operations
  • JWT - Authentication using JSON Web Tokens
  • OpenAI API - For question-answering functionality
  • ChromaDB - Vector database for embeddings storage
  • BeautifulSoup - Web scraping and content extraction

üé® Frontend Technologies

  • Next.js 15 - React framework with App Router
  • NextAuth.js - Authentication
  • Tailwind CSS 4 - Styling
  • TypeScript - Type safety
  • Axios - HTTP client for API requests

üîÆ The RAG Magic

When users search or ask questions, their query is embedded using the appropriate model and compared against all their article chunks using cosine similarity in ChromaDB. The system retrieves the most relevant chunks, deduplicates by article ID, and uses an adaptive threshold system to ensure quality context. The selected content chunks are assembled into a comprehensive context and fed to an OpenAI Foundation model along with the user's question, generating detailed answers that cite specific source articles.

Step-by-Step Experience

Experience the journey of transforming web content into intelligent, searchable knowledge

1

Welcome to Your Research Companion

Start your journey with a clean, intuitive landing page designed for researchers who need to organize and query their knowledge effectively. The modern interface welcomes you to a new way of managing research content.

2

Secure User Registration

Create your personalized research vault with secure JWT authentication. Each user gets their own private knowledge base, ensuring your research remains organized and protected while enabling powerful search capabilities.

3

Simply Paste Any URL

The magic begins with a simple URL paste. Whether it's a research paper, article, or blog post, just drop the link and let the system handle the rest. No complex uploads or manual text entry required.

4

Intelligent Content Processing

Watch as BeautifulSoup scrapes and cleans the content, filtering out ads and navigation. The text is chunked into 800-character segments and converted to vector embeddings using OpenAI's text-embedding-3-small model, then stored in ChromaDB for lightning-fast retrieval.

5

Explore Your Knowledge Base

Browse through your collected articles with rich metadata and similarity scores. Each piece of content is intelligently indexed and ready for semantic search. Select any article to dive deeper or prepare for AI-powered questioning.

6

AI-Powered Research Assistant

Ask complex questions and receive detailed, contextual answers. The RAG system finds the most relevant content chunks, feeds them to OpenAI's foundation models, and generates comprehensive responses with source citations. Your personal research companion understands your collected knowledge and helps you discover insights.
